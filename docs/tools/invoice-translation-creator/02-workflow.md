# Flujo de Trabajo

1. **Ingreso de Información**:
   - El usuario agrega ítems uno por uno a través de un formulario intuitivo.
   - Cada campo del formulario cuenta con ayudas contextuales explicativas.

2. **Revisión**:
   - Los ítems agregados se muestran en una tabla dinámica.
   - El usuario puede editar o eliminar ítems desde esta tabla si detecta errores.

3. **Exportación**:
   - Una vez completado el llenado de todos los ítems, el usuario exporta la información.
   - La herramienta genera un archivo Excel con el formato estándar de traducción de factura requerido por los agentes de aduana.
